# AI Research Writing Skills Hub

[简体中文](README.md) | [English](README_EN.md) | [Bilingual website](https://zejun-w.github.io/AI-Research-Writing-Skills-Hub/)

> A bilingual, researcher-oriented open-source hub that reviews AI paper-writing prompts, agent skills, research methodology repositories, and autonomous research systems, while providing original unified skills that can be adapted to real projects.

## What this repository solves

Research-writing repositories have very different purposes: prompt collections, section-writing skills, scientific toolkits, research methodology notes, and autonomous idea-to-paper systems. A single star ranking hides those differences.

This repository provides four layers:

1. **Continuously updated ranking**: GitHub Actions refresh Stars, forks, issues, licenses, and update dates daily.
2. **Bilingual critical reviews**: Who each project is for, strengths, limits, setup cost, and risks.
3. **Original unified skills**: A practical workflow distilled without copying upstream code.
4. **Copy-ready bilingual prompts**: Paper audit, introduction, related work, method consistency, experiment analysis, review, and Chinese-English transformation.

## Recommendations

- **Primary Codex workflow**: `academic-research-skills-codex` plus this repository's `research-writing-orchestrator`.
- **Computer-science section writing**: `Research-Paper-Writing-Skills` plus `paper-writing-skill`.
- **Chinese prompt material**: `awesome-ai-research-writing`.
- **Experiment-to-paper integration**: `AI-Research-SKILLs`; add `scientific-agent-skills` only when broad scientific tooling is required.
- **Pre-submission QC**: `academic-writing-agents` plus this repository's `paper-qc-reviewer`.
- **Autonomous research systems**: Best treated as research and experimentation platforms, not as a substitute for verified scholarship.

## Use the unified skills

Copy only the skills needed for the current task into your agent's project-level skills directory. Exact directories vary by client and should be checked against official documentation.

```bash
mkdir -p .agent/skills
cp -r skills/research-writing-orchestrator .agent/skills/
cp -r skills/evidence-and-citation-auditor .agent/skills/
cp -r skills/paper-qc-reviewer .agent/skills/
```

Loading only 1–3 relevant skills reduces context noise and conflicting instructions.

## Quality and research-integrity principles

- AI assists organization, checking, and expression; authors remain responsible for facts, experiments, and conclusions.
- Never fabricate papers, DOIs, authors, data, experiments, or reviews.
- Verify citations against primary sources; search snippets are not substitutes for reading papers.
- Separate source facts, inferences, recommendations, and unverified claims.
- Re-run and trace all code, data, figures, and conclusions produced by autonomous research systems.
