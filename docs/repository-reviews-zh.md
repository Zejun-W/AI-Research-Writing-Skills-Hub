## [binary-husky/gpt_academic](https://github.com/binary-husky/gpt_academic)

**定位：** 面向中文科研用户的综合学术助手，强调论文阅读、翻译、润色、LaTeX/PDF处理与插件扩展。

**最适合：** 需要图形界面、中文生态和大量现成学术插件的研究者。

**优势：** 中文使用门槛低；插件生态丰富；适合日常论文处理。

**局限：** 系统较重，配置项多；不等同于严格的人机协同论文工作流。

**兼容/形态：** Web UI, Python, Plugins

**本仓库建议分：** 4.7/5
