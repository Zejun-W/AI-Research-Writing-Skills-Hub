# Contributing / 贡献指南

## 中文

1. 新增项目必须与科研写作、科研工作流、科学Agent或自动科研直接相关。
2. 优先提交上游官方 GitHub 地址，不接受只指向转载站的条目。
3. 总结必须包含：定位、适合用户、优势、局限、主要风险和许可证状态。
4. 不直接复制上游 README 的大段文字；请使用自己的语言概括。
5. 中英文内容应尽量同步。
6. 不以 Stars 作为唯一质量判断，应同时考虑维护状态、方法严谨性和安全性。

## English

1. New entries must directly relate to research writing, research workflows, scientific agents, or autonomous research.
2. Prefer official upstream GitHub repositories over mirrors or directory pages.
3. Each review must include positioning, target users, strengths, limitations, major risks, and license status.
4. Do not copy large sections of upstream READMEs; summarize in original language.
5. Keep Chinese and English content aligned where practical.
6. Stars are not the sole quality signal; maintenance, methodological rigor, and security also matter.
