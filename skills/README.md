# Unified Skills / 统一 Skills

These skills are original workflow abstractions created for this repository. They summarize common best practices without copying upstream skill text.

- `research-writing-orchestrator`: complete human-in-the-loop paper workflow.
- `evidence-and-citation-auditor`: claim, source, and citation integrity.
- `paper-section-rewriter`: bounded section rewriting.
- `paper-qc-reviewer`: pre-submission multi-dimensional review.

Use only the skills required by the current task. Review every SKILL.md before granting an agent file or execution permissions.
